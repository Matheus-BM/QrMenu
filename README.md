* Você instalou a versão mais recente do node.js

## 🚀 Instalando <QrMenu>

Para instalar o <QrMenu>, siga estas etapas:

Linux e macOS:
```
<comando_de_instalação>
```

Windows:
```
git clone https://github.com/Matheus-BM/QrMenu.git;
```
```
cd servidor 
npm install
cd .. 
cd client
npm install
```

## ☕ Usando <QrMenu>

Para usar <QrMenu>, siga estas etapas:

Client 
```
cd client
npm start
```
Server
```
cd server
nodemon index.js
```
